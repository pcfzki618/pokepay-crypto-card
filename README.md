## Limitations and What to Watch Out For

No card is perfect, and pretending otherwise doesn't help you make a real decision. Here's the unvarnished list of PokePay's weak spots:

- **Stacking fees add up.** 1% top-up + 1% transaction + 1% FX = up to 3% effective cost on cross-border spend. Fine for light use, painful for heavy use.

- **Lower daily limits than high-volume alternatives.** ~$12,500/day is fine for consumers, tight for anyone moving meaningful money.
- **Physical card is relatively expensive.** $88 (or ~$70 with promo) is on the higher end — Bybit's physical card costs $5, SafePal's is free.
- **iOS app is region-limited** in some markets, which can frustrate users outside the primary US/HK availability zones.

The point isn't that these are dealbreakers — for the right user they're not. The point is that "no staking, no subscription" comes with its own set of trade-offs, and you should know them before you commit.
